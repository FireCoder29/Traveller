# Traveller – A Travel Vlog Website

Traveller is a simple and attractive travel guide website created using HTML and CSS.
This website is focused **only on the Sylhet division of Bangladesh** and highlights
some of its most beautiful tourist destinations. It is a static website hosted using
GitHub Pages.

### Live Website: 
https://firecoder29.github.io/Traveller/

---

## Features
- Displays popular tourist destinations of Sylhet.
- Clean and simple user interface.
- Image-based destination cards.
- “Learn More” buttons for better navigation.
- Responsive layout for different screen sizes.
- Fast loading static website.

---

## Sections of the Website
- Home section with navigation menu
- Destination section showcasing Sylhet tourist spots
- About section explaining the website purpose
- Footer section with basic information

---

## Tourist Places Highlighted (Sylhet)
- Jaflong
- Ratargul Swamp Forest
- Lalakhal
- Bichanakandi

Each destination includes an image and a short description to give visitors a quick
overview of the place.
